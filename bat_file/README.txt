Open text editor

1.Then set jdk bin path

->set path=C:\Program Files\Java\jdk-10.0.2\bin

2.write compile code in editor.
->javac Test.java

3. Write run java class file code in editor.
->java Test

4. If you don't write pause in bat file then the bat file will execute java file and the command line window will be closed immediately after it shows output.To remain open command line window after successfully compiling please write pause in bat file.

->pause

5.After that save the file as any name but in .bat extension. In this case it is xyz.bat.

6. Double click the bat file to execute the command.
